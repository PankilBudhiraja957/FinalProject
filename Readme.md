# Final Project – Simple Interest Calculator

This repository contains the required files for the Final Project submission. The main script in this project is `simple-interest.sh`, which calculates the simple interest based on user-provided principal amount, rate of interest, and time period.

## Files Included

- `simple-interest.sh` – A shell script that calculates simple interest.
- `LICENSE` – Contains the Apache 2.0 License for this project.
- `CODE_OF_CONDUCT.md` – Defines standards for contributing and interacting in this project.
- `CONTRIBUTING.md` – Provides guidelines for contributing to this repository.
- `README.md` – This file, providing an overview of the project and repository structure.
