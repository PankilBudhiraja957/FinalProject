# Contributing Guide

Thank you for your interest in contributing to this project!

## How to Contribute

1. Fork the repository to your GitHub account.
2. Create a new branch for your feature or fix.
3. Make your changes and test them properly.
4. Submit a pull request (PR) describing what you have changed.

## Reporting Issues

If you find a bug or have a suggestion, please open an Issue with a clear description so it can be reviewed properly.

## Code Style

- Write clean and readable code.
- Add comments where necessary.
- Follow standard Bash scripting practices for the simple-interest.sh script.

## Code of Conduct

This project follows the rules described in the CODE_OF_CONDUCT.md file.
