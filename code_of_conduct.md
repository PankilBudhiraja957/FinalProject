# Code of Conduct

## Our Pledge
We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards
- Be respectful and inclusive.
- Use welcoming and professional language.
- Avoid insulting, derogatory, or discriminatory comments.

## Unacceptable Behavior
- Harassment, bullying, or discriminatory actions.
- Sharing private information without permission.
- Any behavior that makes others feel unsafe.

## Enforcement
Project maintainers may remove or edit comments, pull requests, or issues that violate this Code of Conduct.

## Contact
If you experience unacceptable behavior, please contact the project maintainer.
